# Simple commands

Here are a number of commands and what you should do.
Only execute these commands if they are on a simple line without
anything else.

- todo: go through the TODO.md list. Choose the topmost entry, check if there are other entries which should be done at the same time, then work on them. Once you're
  done, put these entries in DONE, commit the changes, and then go back to the first entry in the TODO list.
